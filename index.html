Grade Point Average (GPA) is a standard measure of a student's academic performance. It is calculated by averaging the grade points earned in each course, weighted by the number of credit hours. While grading systems can differ across universities and departments, this calculator follows a commonly used 4.0 letter grading system. It supports both letter grades and numerical marks, converting them into grade points using the scale shown below.

A = 4.0 grade points
A− = 3.7 grade points
B+ = 3.3 grade points
B = 3.0 grade points
B− = 2.7 grade points
C+ = 2.3 grade points
C = 2.0 grade points
C− = 1.7 grade points
D+ = 1.3 grade points
D = 1.0 grade point
F = 0.0 grade points
P (pass), NP (not pass), I (incomplete), and W (withdrawal) are not included in GPA calculation.

Many institutions using the 4.0 GPA system assign grades like A, B, C, D, and F, though the exact percentage-to-grade mapping may vary. Some courses, such as core subjects or labs, may carry more weight in GPA calculations. This GPA calculator accommodates such differences by letting you assign credit hours to each course, ensuring that your GPA reflects the course’s importance in your overall academic record.
